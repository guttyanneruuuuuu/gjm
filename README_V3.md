# Inverse Hunter V3 - 食物連鎖を逆走する3DアクションRPG

高品質な3Dアクションゲーム「Inverse Hunter」の完全リニューアル版です。スマホとPC両対応で、操作性、ゲームバランス、UIデザインを大幅に改善しました。

## 🎮 ゲーム概要

**コンセプト：** 「弱者が連鎖を昇り、やがて守る者になる」

ネズミとしてスタートし、敵を倒すたびにその力を「食らい、変身する」。ワシになり、ヘビになり、キツネになる。しかし頂点に立った瞬間、新たな敵が現れる。強くなることの意味を問うゲーム。

## 🎯 主な改善点

### ✅ 操作性の大幅改善
- **矢印キーの修正：** 逆向きだった矢印キーを完全に修正
- **スマホ対応：** 完全なタッチコントロール実装
  - 左下ジョイスティック：移動
  - 右下アクションボタン：攻撃、ジャンプ、ダッシュ、スキル
  - 上部形態ホイール：形態切り替え
- **PC対応：** WASD/矢印キー、マウス視点、クリック攻撃

### ✅ ゲームバランスの改善
- **プレイヤーHP：** 100 → 150（大幅増加）
- **敵のHP・攻撃力：** 難易度に応じて自動調整
- **敵の攻撃間隔：** 1.5秒～2.5秒（適切に設定）
- **難易度システム：** Easy / Normal / Hard / Insane
- **「一瞬で負ける」問題を解決**

### ✅ UI/UXの完全リニューアル
- **レスポンシブHUD：** スマホ・PC両対応
- **見やすいステータスバー：** HP、スタミナ、本能ゲージ
- **ボスヘルスバー：** ボス戦時に自動表示
- **コンボカウンター：** 連続攻撃を視覚化
- **ダメージ表示：** 浮き上がるダメージ数字

### ✅ デザインの大幅改善
- **形態ごとの色分け：** 視認性向上
  - 🐭 ネズミ：茶色
  - 🦊 キツネ：橙色
  - 🐍 ヘビ：緑色
  - 🦅 ワシ：黄色
  - 🐺 オオカミ：灰色
- **パーティクルエフェクト：** 攻撃、ダメージ、変身、死亡
- **スムーズなカメラ：** 追従とロックオン
- **敵のAI改善：** アイドル、パトロール、チェイス、攻撃、逃走

## 🎮 操作方法

### PC版
| 操作 | キー |
|------|------|
| 移動 | WASD / 矢印キー |
| 視点操作 | マウス（クリックで固定） |
| 攻撃 | マウス左クリック |
| スキル | マウス右クリック |
| ジャンプ | スペースキー |
| ダッシュ | Shift + スペース |
| 形態切り替え | 1-5キー / E / Q |
| ロックオン | L キー |
| スプリント | Shift キー |

### スマホ版
| 操作 | タッチ |
|------|--------|
| 移動 | 左下ジョイスティック |
| 視点操作 | 右側スワイプ |
| 攻撃 | 赤いボタン（右下） |
| ジャンプ | 緑いボタン（右下） |
| ダッシュ | 黄色いボタン（右下） |
| スキル | 青いボタン（右下） |
| 形態切り替え | 上部の形態ホイール |

### デバッグキー
| キー | 機能 |
|------|------|
| P | ポーズ/再開 |
| R | ゲーム再開 |
| U | 全形態アンロック |
| D | 難易度変更 |

| タイプ | HP | 攻撃力 | 速度 | 経験値 |
|--------|-----|--------|------|--------|

| Medium | 50 | 12 | 10 | 30 |
| Large | 100 | 20 | 9 | 60 |

### 難易度設定
- **Easy：** 敵HP 70%、敵攻撃力 60%、プレイヤーHP 130%
- **Normal：** 標準設定
- **Hard：** 敵HP 130%、敵攻撃力 130%、プレイヤーHP 90%
- **Insane：** 敵HP 180%、敵攻撃力 180%、プレイヤーHP 70%

## 🚀 実行方法

### ブラウザで実行
```bash
# ローカルサーバーを起動
python -m http.server 8000

# ブラウザで開く
http://localhost:8000/index_v3.html
```

### 開発環境
```bash
# 依存関係をインストール
npm install

# 開発サーバーを起動
npm run dev

# ビルド
npm run build
```

## 🎓 ゲームプレイのコツ

1. **形態を使い分ける：** 各形態に得意な敵がいます
2. **ロックオンを活用：** L キーでロックオンして敵を追跡
3. **ダッシュで回避：** Shift + スペースでダッシュして敵の攻撃を避ける
4. **コンボを稼ぐ：** 連続攻撃でダメージを増加させる
5. **スタミナ管理：** スプリントはスタミナを消費するため注意
