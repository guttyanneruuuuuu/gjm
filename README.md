# Inverse Hunter
食物連鎖を逆走する、生存と革命の3Dアクション (Three.js / single-file)

`index.html` 1ファイルだけで動きます。ブラウザで開くだけ。PC・スマホ横画面対応。

## 操作方法 / Controls

### PC
| 操作 | キー |
|---|---|
| 移動 | `W A S D` または 矢印キー `↑ ↓ ← →` |
| 視点 / 照準 | マウス（画面クリックで照準ロック / Esc で解除） |
| 攻撃 | 左クリック または `J` |
| スキル | 右クリック または `K` |
| ジャンプ / 上昇(ワシ) | `Space` |
| 下降(ワシ) | `Ctrl` / `C` |
| ダッシュ | `Shift` |
| 敵ロックオン切替 | `Q` |
| 本能（オオカミは遠吠え回復） | `E` |
| 形態切替 | `1`〜`5` |

### スマホ（横画面）
- 左スティック … 移動
- 画面右側をドラッグ … 視点・照準
- `攻撃` / `スキル` / `跳`(ジャンプ) / `本能` / `ロック` ボタン

## 主な特徴
- **照準（エイム）可能な攻撃**：近接は照準方向への扇状ヒット、ヘビは狙った方向に毒弾、オオカミは遠吠え衝撃波。
- **ロックオン**：対象に自動で向き直り、安定したアクション戦闘。
- **ダッシュ**で素早く間合いを詰める/離す。
- **ボスごとに異なるAI**（追跡 / 射撃カイト / ワシの急降下 / 最終ボスの旋回射撃）と専用HPバー。
- **コンボ・カメラシェイク・被弾ヴィネット**などの手応えあるフィードバック。
- 変身チェーン：🐭 → 🦊 → 🐍 → 🦅 → 🐺 →（🧍 → ⚡）
